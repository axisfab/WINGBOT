## 📌 Overview
본 플랫폼은 Edge AI 기반 자율주행 로봇 개발과 Physical AI 교육을 위한 통합 솔루션입니다.  
연구, 교육, 실전 경진대회까지 활용 가능한 **All-in-One AI 로봇 플랫폼**입니다.

---

## 🎯 Key Features
### 🚀 Edge AI 기반 자율주행
- 실시간 영상 처리 및 객체 인식
- 자율 주행 및 장애물 회피
- 센서 융합 기반 판단 시스템
### 🧠 AI 교육 최적화 플랫폼
- Python 기반 AI 학습 환경
- Jupyter Notebook 실습 지원
- 초급 ~ 고급 단계별 커리큘럼 구성 가능
### 🛠️ Physical AI 통합 구조
- 하드웨어 + 소프트웨어 통합 설계
- 실제 로봇 기반 실습 가능
- 산업/국방형 시나리오 대응
### 🏁 경진대회 및 연구 활용
- AI 자율주행 대회 대응
- 국방/로봇 경진대회 플랫폼 활용 가능
- 실전 문제 해결 중심 설계
