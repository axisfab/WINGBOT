## 📌 Overview
WINGBOT은 Edge AI 기반 자율주행 로봇 개발과 Physical AI 교육을 위한 통합 플랫폼입니다.  
연구, 교육, 실전 경진대회까지 대응 가능한 **All-in-One Autonomous Robotics Solution**입니다.

## ⚙️ Specifications

| Category | Item | Description |
|----------|------|------------|
| 🏗️ Physical | Weight | Approx. 2.5 kg |
| 🏗️ Physical | Structure | Tracked (Tank-type) Robot |
| 🧠 AI Board | Model | NVIDIA Jetson Orin Nano |
| 🧠 AI Board | GPU | 1024 CUDA + 32 Tensor |
| 🧠 AI Board | CPU | 6-Core Cortex-A78AE |
| 👁️ Vision | Camera | AI Camera Module |
| 👁️ Vision | Features | Object Detection / Autonomous Driving |
| 🔌 Connectivity | Interface | USB / UART / GPIO |
| 🔌 Connectivity | Communication | ROS2 |
| 🔋 Power | Battery | 12V Battery |
| 🧩 Expandability | Features | Sensor / IoT / AI 확장 |
